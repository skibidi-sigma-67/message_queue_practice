# Ознакомительная практика, кейс #5

Сравнение стратегий поведения очереди сообщений в
многопоточном in-memory брокере на различных
паттернах нагрузки

### Описание

Цель — определить оптимальную стратегию поведения очереди сообщений для различных
сценариев работы на основе объективных метрик, включая пропускную способность,
латентность операций, использование памяти и fairness между писателями и читателями.
